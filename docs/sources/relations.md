## Description

Honest news-activity index computed from every GDELT-indexed event in the last 24 hours. For each country pair and each country we count distinct cooperative vs conflict events (QuadClass 1+2 vs 3+4) and report the ratio. This is NOT a diplomatic relations matrix — GDELT's auto-coded Goldstein scale is too noisy at the pair level to support that claim — but it is a reliable measure of what kind of news is being written about each country.
